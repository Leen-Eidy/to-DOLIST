# Productivity Dashboard

A **responsive productivity web app** that combines a task manager, Pomodoro-style focus timer, and motivational carousel to help users stay organized, focused, and inspired.  
Built with **HTML5**, **CSS3**, and **JavaScript** for smooth interactivity and modern design.
